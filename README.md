# Ding-install-script
# Hysteria2 Install Script

A simple Hysteria2 server installation script with multi-user support.

## Features

- Install Hysteria2
- Generate self-signed certificate
- Configure multi-user (userpass)
- Setup systemd service
- Open UDP port automatically
- Generate client config and share link

## Usage

```bash
wget -O install.sh https://raw.githubusercontent.com/DingShaoyu/Ding-install-script/main/install.sh
bash install.sh
